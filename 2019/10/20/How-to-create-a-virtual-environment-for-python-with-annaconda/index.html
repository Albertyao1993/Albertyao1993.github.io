How to create a virtual environment for python with anaconda

Anaconda is a platform which can provide a various virtual development environment for date science. People can create different development file. This text recording the approach that using conda to create a specific development environment.

  1. You need to install anaconda on your system. Man can download the install programm on website. www.anaconda,com. Then using command conda -v to check out.

  2. Create a virtual enviroment.
    conda create -n projectname python=x.x anaconda

    tip: command conda search "^python$" can check which python version you can use for your enviroment. Usually we use python3.X to development.

  3. Active yout virtual enviroment.
    source activate projectname

  4. Install additional Python packet,which is necessary for development.
    conda install [packagesname] OR
    pip install [packagesname] OR
    conda install -n projectname [packagesname]

    When you will, you can also install package in pycharm with package manager.

  5. When you don’t need your current enviroment. Use command
    source deactivate
    to close enviroment.

  6. Delete a no longer needed enviroment.
    conda remove -n projectname -all
