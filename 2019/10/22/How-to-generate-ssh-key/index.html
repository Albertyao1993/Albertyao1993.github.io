How to generate ssh key

How to generate a SSH Key on laptop

  1. In terminal input command ls -al ~/.ssh,if you have already generate a ssh key on laptop, this command will return 2 id_rsa and id_rsa.pub file, otherwise it will return No such file or directory

  2. Generate a new SSH_Key
    Use command ssh-keygen -t rsa -C "email@example.com"
    email@example,com is your email address, which you use to register github account.

  3. ssh-add ~/.ssh/id_rsa. After success terminal dispaly
    Identity added: /home/xxx/.ssh/id_rsa (/home/xxx/.ssh/id_rsa)

  4. finally input command cat /home/xxx/.ssh/id_rsa
