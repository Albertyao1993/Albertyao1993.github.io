How to create a virtual environment for python with anaconda

How to create a virtual environment for python with anaconda

Anaconda is a platform which can provide a various virtual development environment for date science. People can create different development file. This text recording the approach that using conda to create a specific development environment.

  1. You need to install anaconda on your system. Man can download the install programm on website. www.anaconda,com. Then using command conda -v to check out.

  2. Create a virtual enviroment.
    conda create -n projectname python=x.x anaconda

    tip: command conda search "^python$" can check which python version you can use for your enviroment. Usually we use python3.X to development.

  3. Active yout virtual enviroment.
    source activate projectname

  4. Install additional Python packet,which is necessary for development.
    conda install [packagesname] OR
    pip install [packagesname] OR
    conda install -n projectname [packagesname]

    When you will, you can also install package in pycharm with package manager.

  5. When you don’t need your current enviroment. Use command
    source deactivate
    to close enviroment.

  6. Delete a no longer needed enviroment.
    conda remove -n projectname -all

Github学习笔记一

Github 学习知识点记录

Create a local repository

  1. first create a empty file for repository
    `
    mkdir learngit
cd learngit
    `
  2. initialization this file for git repository
    `
    git init
    `
  3. create a testfile and use command ‘’’git add’’’ to add this testfile into repository
    `
    git add readme.txt
git commit -m "write a testfile"
    `

m means a discription for testfile.

when you commt,sometimes system maybe remind you:Your name and email address were configured automatically based
on your username and hostname. Please check that they are accurate.
You can suppress this message by setting them explicitly. Run the
following command and follow the instructions in your editor to edit
your configuration file:

it happend especially in MBP user.

solution
git config --global user.name "Your Github username" git config --global user.email "Your Github email address" git commit --amend --reset-author.

How to create a remote repository on Github and associate with local repository?

  1. You need to create a new repo on Github
  2. use command
    git remote add origin git@github.com:Albertyao1993/learngit.git
  3. push your file to github
    git push -u origin master

tip:
when you first time push your code, command git push -u origin master .
later you can usegit push orgin master
